# Week 2 — Challenges

1. **[Challenge 1 — Shrink to under 50 MB](challenge-01-shrink-to-under-50mb.md)** — Take a moderately complex Python app to a < 50 MB image while keeping every runtime requirement intact. Distroless allowed; cheating with `FROM scratch` not allowed (Python needs an interpreter). (~3 hours)
