# Week 4 — Challenges

Challenges are scoped tighter than exercises and looser than the mini-project. Each one has a single, measurable outcome and an opinionated path to get there. They are graded by whether the end state works, not by the path you took.

Saturday is challenge day. One challenge, two coffees, an afternoon of focused work. The success criterion is a green workflow run and a working `docker pull` of the resulting image from a clean machine.
